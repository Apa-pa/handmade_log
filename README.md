# ハンドメイド・ログ (Handmade Log)

ハンドメイド作家のための資材管理および作品別材料費計算ツールです。

## プロジェクト概要
このアプリケーションは、ハンドメイド作家が所有する資材（パーツや布など）を管理し、それらを使用して制作した作品の正確な材料費を簡単に算出できるようにすることを目的としています。

### 技術スタック
- **Frontend**: HTML5, CSS3, JavaScript (Vanilla JS)
- **Design**: モバイルファースト (iPhone 向けデザイン)、PWA 対応
- **Data Storage**: 
  - 初期段階: ブラウザの `LocalStorage`
  - 将来的拡張: `Firebase` によるクラウド同期・バックアップ

## 主な機能

### 1. 資材管理 (Material Management)
- **登録項目**:
  - 資材名
  - 金額 (必須)
  - 単位 (個、m、gなど) (必須)
  - 購入日 (任意)
  - 店舗名 (任意)
  - 備考 (任意)
- **一覧表示**: 登録済みの資材を確認、編集、削除が可能。

### 2. 作品登録・材料費計算 (Work Registration & Cost Calculation)
- **作品登録**:
  - 作品名
  - 使用資材の選択（登録済み資材から選択）
  - 使用量の入力
  - 販売価格 (任意)
  - 備考 (任意)
- **成約機能**: 作品が売れた際に「成約」としてマークし、統計に反映。
- **自動計算**: 選択された資材の単価と使用量から、作品ごとの合計材料費をリアルタイムに算出。

### 3. 経費管理 (Expense Management)
- **登録項目**:
  - 用途/名目
  - 日付
  - 金額
  - 備考
- 材料費以外の諸経費（梱包材、送料、広告費など）を記録。

### 4. 成約済み作品リスト & 統計
- 成約した作品の一覧表示。
- 月ごとの売上合計、材料費合計、経費合計、利益（売上 - 材料費 - 経費）の自動集計。

### 5. トップ画面
- アプリ起動時のメインメニュー。
- 「資材登録」、「作品登録」、「経費登録」、「成約済みリスト」の主要な導線を提供。

## デザインコンセプト
- **テーマカラー**: 極めて淡い紫 (#F8F6FB 周辺) を基調とした、清潔感と上品さのあるデザイン。
- **UI/UX**: 
  - iPhone での操作を前提とした親指で操作しやすいレイアウト。
  - PWA 化により、ホーム画面からネイティブアプリのような感覚で起動可能。
